Start.
Create a priority queue and insert the start node with cost 0.
Create an empty visited set (or dictionary) to store explored nodes.
Repeat until the priority queue is empty:
Remove the node with the minimum cumulative cost from the priority queue.
If the removed node is the goal node, display the path and total cost, then stop.
If the node has not been visited, mark it as visited.
For each neighboring node:
Calculate the new cumulative cost = current cost + edge cost.
Insert the neighbor into the priority queue with the new cost.
If the goal node is not found after the queue becomes empty, print "Path not found."
Stop.
